Về Lĩnh Vực Mạng Máy Tính

Mạng máy tính là "hệ thần kinh" của mọi tổ chức trong kỷ nguyên số. Việc hiểu rõ cách các gói tin di chuyển, cách định tuyến (Routing) tối ưu và ngăn chặn các mối đe dọa bảo mật là yếu tố sống còn để doanh nghiệp vận hành trơn tru.

Tôi tập trung vào việc nghiên cứu kiến trúc mạng LAN/WAN, các giao thức định tuyến tĩnh và động, cũng như việc triển khai các ứng dụng thực tế trên môi trường Docker để tối ưu tài nguyên phần cứng.

Thiết kế mạng thực tế

Phân hoạch mạng con (Subnetting), thiết kế mạng LAN nội bộ an toàn.

Bảo mật hệ thống

Cấu hình Firewall, phân quyền kiểm soát truy cập (Access Control List - ACL).

Môi Trường & Công Nghệ Sử Dụng

Các công cụ và nền tảng tôi sử dụng để xây dựng và tự động hóa hệ thống.

Docker & Containers

Đóng gói ứng dụng, tối ưu hóa quá trình triển khai dịch vụ mạng trên môi trường ảo hóa nhẹ.

GitHub CI/CD

Tự động hóa quá trình kiểm thử, đóng gói và phát hành ứng dụng lên các nền tảng đám mây.

Linux Administration

Quản trị các dịch vụ mạng (DNS, DHCP, Web Server, SSH) trên hệ điều hành Ubuntu/Debian.

Mô Hình Tham Chiếu Mạng Cơ Bản

Mô Hình 7 Lớp OSI

  • Lớp 7 - Application HTTP, DNS, FTP
  • Lớp 4 - Transport TCP, UDP
  • Lớp 3 - Network IP (IPv4, IPv6), ICMP
  • Lớp 2 - Data Link Ethernet, MAC Address
  • Lớp 1 - Physical Cáp mạng, Sóng Wi-Fi

Giao Thức Quan Trọng

TCP (Transmission Control Protocol)

Giao thức hướng kết nối, đảm bảo dữ liệu truyền đi chính xác, đầy đủ và đúng thứ tự.

IP Routing (Định tuyến IP)

Quy trình chọn đường đi tối ưu nhất cho gói tin đi qua các Router để đến được đích.

DNS (Domain Name System)

Dịch vụ phân giải tên miền (như github.com) thành địa chỉ IP để máy tính có thể hiểu được.
